repeats

repeats: number
  • Defined in types.ts:33

Number of times the error has been repeated (starts with 0).

It helps detecting when an iterator starts throwing the same error in a loop, so a different error-handling strategy can be applied, such as (for example) re-throwing only when the error is repeated.

.catch((e, ctx) => {
    if(ctx.repeats) {
        throw e; // re-throw when repeated
    }
    console.log(e?.message || e); // report the error
});
